 Tic Tac Toe Web App

An interactive and responsive Tic Tac Toe game built using **HTML, CSS, and JavaScript**. This project demonstrates core frontend development skills including DOM manipulation, game logic implementation, and UI design.

* 🧠 Two-player game logic
* 🏆 Win detection system
